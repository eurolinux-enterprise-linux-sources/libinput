Detailed Description

Tablet pad event representing a button press, or ring/strip update on the tablet pad itself.

Valid event types for this event are LIBINPUT_EVENT_TABLET_PAD_BUTTON, LIBINPUT_EVENT_TABLET_PAD_RING and LIBINPUT_EVENT_TABLET_PAD_STRIP.
